The Brief

WHERE
Type Lives

Designers spend hours hunting for the right typeface across fragmented platforms — downloading, installing, testing, comparing. Fontify was designed to collapse that workflow into a single, elegant mobile experience.

The design challenge: an app about type must itself be typographically exceptional. Every interaction had to feel intentional, every screen had to demonstrate what the app celebrates.

Key Features
01
LIVE PREVIEW

Type your own text and see it rendered in any font, at any size, in real time. No more mental leaps between specimen and reality.

02
PAIR SUGGEST

AI-assisted pairing recommendations based on visual rhythm, contrast, and historical typographic convention.

03
FULL SPECIMEN

Every weight, every glyph, every language character. Detailed metrics including x-height, cap height, and optical size.

04
INSTANT INSTALL

One tap to install fonts system-wide on iOS. Available immediately in every app — Figma, Notes, Pages, anywhere.

Design System

TYPE
TEACHING
Itself

The Fontify design system is built on a single principle: the interface must demonstrate what it teaches. Every type choice in the UI is intentional and documented — the app is its own specimen.

Scale moves from display to micro with clear purpose. Spacing follows a base-8 grid. Color defers entirely to the typefaces being shown.
